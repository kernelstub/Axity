# Printing

## Syntax
- `print(expr);`
- Evaluates `expr` and writes a formatted value followed by a newline

## Example
## Formatting
- `int`: decimal
- `string`: raw contents
- `bool`: `true` / `false`
- `array`: `[elem1, elem2, ...]`
- `map`: `{key: value, ...}`
- `object`: `Class{field: value, ...}`

## Example
```
let x: int = 42;
print(x);            // 42
print("hi");         // hi
print(true);         // true
print([1,2]);        // [1, 2]
```
